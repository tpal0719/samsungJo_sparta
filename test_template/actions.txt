이 JavaScript 코드는 웹 페이지에서 fullPage 스크롤링과 Swiper 슬라이더를 함께 사용하는 예시입니다. 각각의 라이브러리를 사용하여 스크롤링과 슬라이딩 인터랙션을 통합하며, 서로 간의 상호작용을 관리하는 로직이 포함되어 있습니다. 코드를 부분별로 세부적으로 분석해 보겠습니다.

### 1. 전체 설정 (`$(document).ready`)

`$(document).ready()`는 문서의 DOM이 완전히 로드되고 준비되었을 때 실행됩니다. 이는 JavaScript가 실행되기에 적절한 DOM 환경이 준비되었음을 보장합니다.

### 2. Fullpage 설정

`$('#fullpage').fullpage({...})`은 fullpage.js 라이브러리를 초기화하고, 스크롤링에 대한 다양한 설정을 포함합니다.

- `sectionsColor`: 각 섹션의 배경색을 배열로 지정합니다.
- `anchors`: 각 섹션에 할당된 앵커(고유 이름)를 배열로 지정합니다.
- `menu`: 섹션과 연결될 메뉴의 ID를 지정합니다.
- `scrollingSpeed`: 섹션 간 전환 속도를 밀리초 단위로 설정합니다.
- `onLeave`: 섹션을 떠날 때 실행되는 함수로, 스크롤, 터치, 마우스 휠 이벤트를 차단하여 다중 이벤트 발생을 방지합니다. 또한 `swiper.mousewheel.disable()`를 호출하여 Swiper 슬라이더의 마우스 휠 제어를 비활성화합니다.
- `afterLoad`: 섹션이 로드된 후 실행되는 함수로, 이벤트 차단을 해제합니다. 특정 조건(예: 활성 섹션이 'sec2'가 아닐 경우)에서 fullpage 스크롤을 다시 허용합니다.

### 3. Swiper 설정

`var swiper = new Swiper('.swiper-container', {...})`은 Swiper 슬라이더 라이브러리를 초기화하고 설정합니다.

- `slidesPerView`: 한 번에 보여줄 슬라이드 수입니다.
- `spaceBetween`: 슬라이드 간의 간격을 설정합니다.
- `freeMode`: 자유롭게 슬라이드 할 수 있는 모드를 활성화/비활성화합니다.
- `speed`: 슬라이드 전환 속도를 설정합니다.
- `pagination`: 페이지네이션 관련 설정으로, 클릭 가능 여부 등을 설정합니다.
- `mousewheel`: 마우스 휠로 슬라이드를 제어할 수 있도록 설정합니다.
- `on`: 슬라이더 이벤트에 대한 핸들러를 정의합니다.
  - `slideChange`: 슬라이드가 변경될 때 호출되며, 특정 슬라이드에서는 fullpage의 스크롤을 차단합니다.
  - `slideChangeTransitionEnd`: 슬라이드 전환이 완료된 후 호출되며, 처음과 마지막 슬라이드에서는 fullpage의 스크롤을 다시 허용합니다.
  - `touchMove`: 터치 이동 시 슬라이드를 다음 또는 이전으로 자동 전환하도록 처리합니다.

### 4. Swiper와 Fullpage의 상호 작용

- Swiper 슬라이더 내에서 슬라이드 변경(특히 섹션2 내에서)에 따라 fullPage.js의 스크롤 허용 여부를 동적으로 제어합니다. 이는 사용자가 슬라이더를 조작하는 동안 불필요한 페이지 스크롤을 방지하기 위함입니다.
